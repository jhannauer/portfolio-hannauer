# Portfolio

Portfolio personnel présentant mes projets réalisés du lycée jusqu’à aujourd’hui.

## Ce que j’ai fait
- Migration des pages statiques vers une structure PHP plus simple
- Centralisation du header et du footer
- Mise en place d’un rendu dynamique des projets
- Intégration de Tailwind CSS
- Gestion de l’affichage des médias, images et vidéos
- Ajout d’un système de cartes consultées/non consultées

## Technologies utilisées
- PHP
- HTML
- Tailwind CSS
- JavaScript
- Font Awesome

## Installation / lancement
Pour consulter le site, il suffit d’un serveur PHP local comme XAMPP.

1. Placer le dossier dans htdocs/ et accéder via localhost/www/

2. Installer les dépendances
   npm install

3. Compiler Tailwind
   npx tailwindcss -i ./assets/input.css -o ./assets/tailwind.css --watch
